# 🚀 OPML Gardener - 快速开始

## 第一步：安装依赖

在项目根目录运行以下命令：

```bash
npm install
```

或者如果您使用 yarn：

```bash
yarn install
```

或者使用 pnpm：

```bash
pnpm install
```

## 第二步：启动开发服务器

```bash
npm run dev
```

您会看到类似这样的输出：

```
  ▲ Next.js 14.x.x
  - Local:        http://localhost:3000
  - Network:      http://192.168.x.x:3000

 ✓ Ready in 2.3s
```

## 第三步：打开浏览器

访问 [http://localhost:3000](http://localhost:3000)

## 第四步：测试应用

1. 您可以使用项目中的 `example.opml` 文件进行测试
2. 拖拽文件到上传区域，或点击选择该文件
3. 开始体验各项功能！

---

## 🎯 主要功能一览

✅ **导入 OPML** - 支持拖拽上传
✅ **在线编辑** - 修改标题、分类、链接
✅ **搜索筛选** - 按关键词或分类筛选
✅ **批量操作** - 批量选择和导出
✅ **去重功能** - 自动删除重复订阅源
✅ **导出功能** - 导出全部或选中的订阅源

---

## 📦 生产环境构建

如果您想要构建生产版本：

```bash
# 构建
npm run build

# 启动生产服务器
npm run start
```

## 💡 技术栈

- **Next.js 14** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **shadcn/ui** - UI 组件库
- **fast-xml-parser** - OPML 解析
